## Como rodar

### Pré-requisitos

- Python 3.10+

### Instalação

```bash
git clone https://github.com/salomaocalheiros/BovTaG.git
cd BovTaG
pip install -r requirements.txt
```

### Execução

```bash
python main.py
```
